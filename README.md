# Цветочный магазин - Модульное приложение

Модульное приложение для управления цветочным магазином. Демонстрирует архитектуру с ядром и подключаемыми модулями.

## Требования

- .NET 9 SDK или выше

## Быстрый старт

```bash
# 1. Клонирование репозитория
git clone https://github.com/Ecolis/flower-shop-modules.git
cd flower-shop-modules

# 2. Сборка проекта
dotnet build

# 3. Запуск приложения
dotnet run --project FlowerShop.App

# 4. Запуск тестов
dotnet run --project FlowerShop.Tests
